# Grade Analyzer

A Python project that analyzes student grades and generates a report card.


## Concepts Used
- Lists, Dictionaries, Tuples, Sets
- Functions
- Loops and Conditionals
- F-strings

## What it does
- Takes a list of students with their grades
- Calculates average score for each student
- Assigns a grade letter (A/B/C/F) based on score boundaries
- Displays a clean report card for each student
- Shows unique courses offered using a set
