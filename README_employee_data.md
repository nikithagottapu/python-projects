# Employee Salary Analysis

A Python project that analyzes employee salary data and identifies employees earning below the average salary.

## Concepts Used

* Lists and Dictionaries
* Functions
* Loops and Conditionals
* Type Conversion
* F-strings

## What it does

* Stores employee details in a list of dictionaries
* Calculates the total and average salary of employees
* Compares each employee's salary with the average salary
* Displays details of employees earning below the average salary
* Formats output in a clean and readable manner

## Skills Practiced

* Data manipulation using Python collections
* Function creation and usage
* Iteration through nested data structures
* Conditional filtering
* Basic data analysis
